# TapRooT® Root Cause Analysis (RCA) Portfolio

Welcome to my TapRooT® Incident Investigation and Root Cause Analysis portfolio. This repository serves as a practical demonstration of my ability to lead and execute comprehensive investigations using the TapRooT® methodology, software, and advanced team leadership techniques.

> **⚠️ Confidentiality Notice:** All data, scenarios, and incident reports contained in this repository are strictly mock scenarios, sanitized historical data, or based on publicly available domain events (e.g., CSB or NTSB reports) to protect corporate confidentiality.

## 🎓 Certifications & Training Applied

This repository showcases the practical application of skills acquired through the following official TapRooT® training tracks:

1. **Self-Guided Online Training: TapRooT® eLearning Pathways**
   - **Focus:** Foundational understanding of the 7-step TapRooT® process, basic SnapCharT® creation, and identifying Causal Factors.
2. **Performing an Investigation Using the TapRooT® Software**
   - **Focus:** Digital translation of the investigation process, utilizing the TapRooT® software to build interactive SnapCharTs®, map findings to the Root Cause Tree®, and generate Corrective Action Helper® reports.
3. **TapRooT® Advanced Root Cause Analysis Team Leader**
   - **Focus:** Leading complex, major incident investigations, conducting cognitive interviews, analyzing equipment failures, and presenting robust findings to executive management.
